# TFM-InteligenciaArtificialAplicada

Este repositorio contiene el código de usado durante el desarrollo de mi Trabajo de Fin de Máster: Sistema multiagente para la simulación de la circulación de autobuses de sustitución de la línea 6 de metro de Madrid. 

El código ha sido desarrollado usando la siguiente versión de GAMA: GAMA Platform - 2025.6.4.
